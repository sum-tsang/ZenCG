# ZenCG - 3D Modelling Web Application

## About

ZenCG is a web-based 3D model editor, offering beginners an accessible and usable experience. Alternatively, intermediate users can use it to edit models on the fly, or educators can use it to demonstrate basic 3D creation using intuitive online software.

### You can view the app and blog by clicking on the links below.

| App | Link |
| - | - |
| Home | https://zencg.onrender.com
| Editor | https://zencg.onrender.com/editor/ |
| Blog | https://zencg.onrender.com/blog/ |


## How To Run (Local)

1. Ensure you are in the `~/projects/2026-csc1049-ebrennan-projectmanager/code` repository.
2. Run `python3 -m http.server 8000` in the terminal.

View the local instance (home, editor, and blog respectively) here:

- http://localhost:8001/
- http://localhost:8001/editor/
- http://localhost:8001/blog/
