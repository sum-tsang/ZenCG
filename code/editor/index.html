      User Manual

      ZenCG Quick Guide

      A fast, friendly walkthrough for importing, transforming, and exporting your models.

      Getting Started

      Import an OBJ, select it in the Objects list, then use the transform tools to move, rotate, or scale.

      Tip: Base mesh objects auto-scale to keep proportions consistent.

      Transform Tools

      Use the Mode buttons to switch between translate, rotate, and scale. Numeric inputs allow precise edits.

      Shortcuts: G = Move, R = Rotate, S = Scale

      Materials

      Adjust Base Color and upload textures. Remove textures anytime to revert to solid color.

      Supports PNG and JPG textures.

      Export & Undo

      Export the selected object or the whole scene. Undo deletes with Cmd/Ctrl + Z.

      Delete: Del / Backspace
