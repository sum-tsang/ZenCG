# ZenCG Code Repository

## How To Run (Local)

Ensure you are in the "~/projects/2026-csc1049-ebrennan-projectmanager/code" repository, and run `python3 -m http.server 8000`, in the terminal.

The site, editor, and blog respectively:

- http://localhost:8001/
- http://localhost:8001/editor/
- http://localhost:8001/blog/

## How To View Deployment

Simply go to https://zencg.onrender.com/, and click on either the editor or the blog. 

Alternatively, you may visit the links here:
- https://zencg.onrender.com/
- https://zencg.onrender.com/editor/
- https://zencg.onrender.com/blog/
